📖 Petunjuk

📌 Cara Menggunakan:

1. Pilih salah satu soal latihan yang tersedia.

2. Baca soal dengan teliti.

3. Pilih jawaban yang paling benar.

4. Gunakan tombol Next dan Previous untuk berpindah soal.

5. Tekan tombol Selesai untuk melihat hasil nilai.

6. Setiap soal memiliki timer 100 menit (bisa dimatikan).

7. Gunakan tombol 📋 Daftar Soal untuk navigasi cepat.

8. Tekan tombol Evaluasi di halaman hasil untuk melihat pembahasan jawaban.

💡 Tips:

• Kerjakan soal dengan tenang dan fokus

• Manfaatkan fitur timer untuk mengatur waktu

• Gunakan fitur evaluasi untuk mengetahui pembahasan lengkap

• Data soal disimpan di file soal.js — edit file tersebut untuk update soal

ℹ️ Tentang

📱 Aplikasi Latihan SKD

Latihan SKD adalah aplikasi latihan soal berbasis HTML, CSS dan JavaScript yang dapat dijalankan secara offline maupun online.

Data soal disimpan di file soal.js yang dapat diedit secara terpisah untuk memudahkan pembaruan soal.

⭐ Fitur Unggulan:

• Pengacakan pilihan jawaban setiap soal dibuka

• Timer 100 menit (dapat dimatikan/dinyalakan)

• Sidebar grid soal untuk navigasi cepat

• Hasil nilai/skor otomatis dengan passing grade

• Evaluasi jawaban lengkap dengan pembahasan detail

• Reset progress untuk memulai dari awal

• Data soal terpisah di file soal.js (mudah diupdate)

📊 Passing Grade:

• TWK: 65 | TIU: 80 | TKP: 166
