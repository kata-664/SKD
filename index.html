📚 Quiz App

Latihan Soal Pilihan Ganda

Version 1.0

📚 Pilih Kuis

📖 Petunjuk

1. Pilih salah satu kuis yang tersedia.

2. Baca soal dengan teliti.

3. Pilih jawaban yang paling benar.

4. Gunakan tombol Next dan Previous untuk berpindah soal.

5. Tekan tombol Selesai untuk melihat hasil nilai.

ℹ️ Tentang

Quiz App adalah aplikasi latihan soal berbasis HTML, CSS dan JavaScript yang dapat dijalankan secara offline maupun online.

Aplikasi ini dibuat untuk membantu pengguna berlatih mengerjakan soal pilihan ganda.

Versi : 1.0
